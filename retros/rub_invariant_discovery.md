# Reflect: Discovering vs Manufacturing Invariants

## Continue
We must continue attacking sloppy vocabulary. The Operator correctly identified the danger in my framing: one cannot "manufacture" or "yield" an invariant. An invariant is an inherent truth of a system; it can only be discovered.

## Start
We must start recognizing that a Probe's function is strictly to drill down to the **root physical reality** of a system. An atomic Probe successfully completes when it isolates the true root cause (the invariant). Only then can an `[EXECUTE]` node be spawned to mechanically enforce that discovered reality.

## Stop
We must stop treating invariants as "rules we decide to enforce" (manufactured constraints). If a Probe stops at a superficial constraint layer, it will spawn brittle execution trails. We must stop hallucinating invariants and strictly discover them.
